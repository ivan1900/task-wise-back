## Description

This is backend application to manage tasks and priorities.

## Project setup

Use node version 24.x

```bash
$ npm install
```

To up dockers run:

```bash
$ make run-daemonized
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
